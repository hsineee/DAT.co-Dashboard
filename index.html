Methodology

How the indicator is built

  1. Download Strategy's daily BTC holdings history.
  2. Download Bitcoin daily USD prices.
  3. Use Strategy's official diluted-share snapshots.
  4. Forward-fill diluted shares between official snapshot dates.
  5. Compute BTC per diluted share and then BTC NAV per diluted share.
